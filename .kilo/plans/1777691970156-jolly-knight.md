# Plan: Fix All MCP Server Configuration Issues

## Problems Identified
All 4 MCP servers in antigravity are failing or slow:

1. **firebase-mcp-server** - Loading slowly (5-10s) due to `@latest` and OAuth errors
2. **cloud-sql-mysql-admin** - Fails: "could not find default credentials"
3. **dart-mcp-server** - Fails: "executable file not found in %PATH%"
4. **cloudrun** - Likely has similar auth issues

## Root Causes
- Config file: `C:\Users\USER\.gemini\antigravity\mcp_config.json`
- Missing Google Application Default Credentials (ADC)
- Missing Dart SDK installation
- firebase-tools using `@latest` causing registry checks on every launch
- OAuth token state corruption (see daemon logs)

## Implementation Order

### Phase 1: Install Required Tools
**Order**: These are independent installations

1. **Install Dart SDK** (for dart-mcp-server)
   - Download installer from https://dart.dev/get-dart
   - Run installer, check "Add to PATH"
   - Verify: `dart --version` in new terminal

2. **Install Google Cloud SDK** (for all Google services)
   - Download from https://cloud.google.com/sdk/docs/install
   - Or: `choco install gcloudsdk` if using Chocolatey
   - Run `gcloud init` after installation

### Phase 2: Authentication Setup
**Depends on**: Google Cloud SDK installed

3. **Set up Application Default Credentials**
   ```bash
   gcloud auth application-default login
   ```
   - Follow browser OAuth flow
   - Creates: `%APPDATA%\gcloud\application_default_credentials.json`
   - Verify: `gcloud auth application-default print-access-token`

### Phase 3: Update MCP Configuration
**Depends on**: Dart and gcloud installed + auth complete

4. **Edit `C:\Users\USER\.gemini\antigravity\mcp_config.json`**:
   - Pin firebase-tools version: line 25 → `"firebase-tools@13.14.0"`
   - (Optional) Add `GOOGLE_APPLICATION_CREDENTIALS` env to Google servers if not using default location

5. **Remove/disable unused servers** (optional, reduces startup overhead):
   - If not using Dart: delete `dart-mcp-server` block
   - If not using Cloud SQL: delete `cloud-sql-mysql-admin` block
   - If not using Cloud Run: delete `cloudrun` block

### Phase 4: Pre-caching & Verification
**Depends on**: Config updated

6. **Pre-cache npm packages** (run separately for each):
   ```bash
   npx -y firebase-tools@13.14.0 mcp
   npx -y @google-cloud/cloud-run-mcp
   npx -y @toolbox-sdk/server@>=0.26.0 --prebuilt cloud-sql-mysql-admin --stdio --user-agent-metadata antigravity
   ```
   - Press Ctrl+C after each starts (just to cache)

7. **Test each server individually** (in separate terminals):
   - Firebase: `npx -y firebase-tools@13.14.0 mcp` → should start without errors
   - Cloud SQL: `npx -y @toolbox-sdk/server@>=0.26.0 --prebuilt cloud-sql-mysql-admin --stdio` → should connect
   - Cloud Run: `npx -y @google-cloud/cloud-run-mcp` → should initialize

8. **Restart antigravity/IDE** and check daemon logs
   - Monitor `C:\Users\USER\.gemini\antigravity\daemon\`
   - Verify each server shows "initialized" without ERROR lines

## Expected Results
- **firebase-mcp-server**: 5-10s → < 1s startup
- **cloud-sql-mysql-admin**: Authenticates successfully, no credential errors
- **dart-mcp-server**: Launches if Dart in PATH, or gracefully disabled
- **cloudrun**: Authenticates successfully

## Quick Fix Alternative
If you only need Firebase and want to skip other installations:

```json
{
  "mcpServers": {
    "firebase-mcp-server": {
      "$typeName": "exa.cascade_plugins_pb.CascadePluginCommandTemplate",
      "command": "npx",
      "args": ["-y", "firebase-tools@13.14.0", "mcp"],
      "env": {}
    }
  }
}
```

Remove `cloud-sql-mysql-admin`, `dart-mcp-server`, and `cloudrun` blocks entirely.

## Rollback Plan
1. Restore original `mcp_config.json` from git if tracked
2. Or revert changes manually
3. `npm cache clean --force` if cache issues

## Trade-offs
- Pinning versions requires manual security updates (~monthly)
- Service account keys need secure storage (but more reliable than user OAuth)
- Installing SDKs adds disk space (~700MB total for Dart + gcloud)
